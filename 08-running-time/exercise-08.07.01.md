# Exercise 8.7.1

> Give the order of the following functions using big O-notation.
> 1. `λ n.  2 n + 13`
> 2. `λ n.  5 n^2 + 2 n + 13`
> 3. `λ n.  5 n^3 + n (n + 1) + 13 n + 2`
> 4. `λ n.  2 n + n^4`
> 5. `λ n.  2^(n + 5) + n^2`

---

The expressions have order of growth `O(n)`, `O(n^2)`, `O(n^3)`, `O(n^4)` and `O(2^n)`.
